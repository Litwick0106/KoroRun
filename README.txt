Script
・CameraAngle
Sphereにカメラを追従させていて、そのままだとカメラも回転してしまうのでカメラは一定の間隔を持ちながら回転させないようにする
・Gamemanager
基本的なシーン遷移をこのスクリプトで行う。シーン間で共有されるようになっている。
シーン遷移したときの剛体の値を初期化している
・GameStart
タイトル画面のボタンでシーン遷移を行う
・PlayerRotate
土台を矢印キーで傾ける。最大角度も決められている
・ScriptTimer
タイマー機能が付いていて、時間を過ぎるとシーン遷移を行う
・StageClear
アタッチされたコライダーにPlayerタグが付いたオブジェクトがぶつかるとシーン遷移する。ゴール用
・TimeCounter
カウントダウンを表示するテキストにアタッチする
・TutorialClear
StageClearと同じ
